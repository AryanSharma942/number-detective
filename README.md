# Digit Detective - Number Guessing Game

A simple and interactive number guessing game built using HTML, CSS, and JavaScript.

## About the Game

Digit Detective is a browser-based game where the player tries to guess a randomly generated number between 1 and 20.

The player has 8 attempts to guess the correct number.

## How to Play

1. Enter a number between 1 and 20
2. Click on "Submit guess"
3. The game will guide you:

   * If your guess is too high
   * If your guess is too low
4. Continue guessing until:

   * You find the correct number
   * Or you run out of attempts

## Features

* Random number generation
* Input validation (only numbers between 1–20 allowed)
* Tracks previous guesses
* Displays remaining attempts
* Real-time feedback (low / high / correct)
* Game over and win detection

## Tech Stack

* HTML – Structure
* CSS – Styling
* JavaScript – Game logic
